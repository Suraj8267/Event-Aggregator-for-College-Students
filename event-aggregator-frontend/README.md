# College Event Aggregator - Frontend

A React frontend for the College Event Aggregator system that allows students to discover, register, and manage college events.

## Features

- **User Authentication** - Register, login, and profile management
- **Event Browsing** - Filter and search events by category, department, and date
- **Event Registration** - One-click registration for events
- **Event Creation** - Organizers can create and manage events
- **Notifications** - Real-time notifications for event updates
- **Responsive Design** - Works on all devices
